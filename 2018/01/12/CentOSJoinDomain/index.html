CentOS加入AD域

2018-01-12
  • 笔记存档

1.安装软件

yum install krb5-libs krb5-devel pam_krb5 krb5-workstation krb5-auth-dialog 
yum install samba-winbind samba samba-common samba-client samba-winbind-clients

2.图像化配置

authconfig-tui

use winbind
use kerberos
use winbind authertication

3.vi /etc/sysconfig/network

HOSTNAME= host.domian.com

hostname test01
service network restart #重启网络。

4.vi /etc/nsswitch.conf

passwd: files winbind
shadow: files winbind
group: files winbind

5.vi /etc/krb5.conf

6.vi /etc/resolv.conf

nameserver 10.2.232.191

7.vi /etc/hosts

10.2.236.205 test01.testad.com test01

8.vi /etc/samba/smb.conf

修改如下一行，可以实现登录时不需要输入域名

winbind use default domain = true

realm = TESTAD.COM

启动相关服务并设置开机自启动，执行如下命令：

service smb start
chkconfig smb on

9.加域

net ads join -U administrator

10.测试加域是否成功

wbinfo -u ##查看域用户
wbinfo -g ##查看域组

测试ntlm组件

ntlm_auth --username=administrator
password: ##输入用户密码
NT_STATUS_OK: Success (0x0)

验证代域

net ads testjoin

11.设置域用户家目录

在/etc/pam.d/system-auth以及/etc/pam.d/sshd文件中的sesson部分添加

session required /lib/security/$ISA/pam_mkhomedir.so skel=/etc/skel umask=0077
